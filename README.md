# U.S. Housing & Land Price Forecasting
Indiana University Indianapolis — Undergraduate Econometric Research

## Overview
This project combines Stata econometrics with Python ARIMA forecasting to analyze U.S. housing prices.

## Files
- data/housing_clean.csv – cleaned dataset
- notebooks/housing_forecast_arima.ipynb – Python ARIMA workflow
- stata/housing_analysis.do – Stata models
- visuals/ – charts
